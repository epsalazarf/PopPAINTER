# PopCanvas

_Auto-PCA Plotter (R Shiny) v1.0_

Developed by: **Pavel Salazar-Fernandez**

## Description

This Shiny application provides an interactive visualization tool for PCA results generated by PLINK. The app reads `.eigenvec` and `.eigenval` files and a formatted `popinfo` file to generate a customizable and interactive PCA plot. The user can zoom into specific regions of the plot, filter populations, and apply different labeling and coloring criteria.

## Version Information

- **Current Major Version**: v1.0  
- **Release Date**: August 20, 2025
- **R Version**: 4.0 or higher required.

## Features

- **Interactive Zoom**: Select and zoom into specific areas of the plot using the brush and double-click.
- **Population Selection**: Choose populations to display or highlight from the provided `popinfo` file.
- **Color Customization**: Use any category in the `popinfo` file to color the PCA points.
- **Dynamic Plot Options**: Choose between points, text, or population labels for plot markers.
- **Save Options**: Download the current plot as PNG or PDF.

## Requirements

- **PLINK PCA Results**: `.eigenvec` and `.eigenval` files from a PCA run.
- **Popinfo File**: A tab-separated `.txt` file that includes information about population, region, and other labels. The first column must match the IDs in the `.eigenvec` file.

## Installation

### Prerequisites

- Ensure you have **R** and **RStudio** installed.
- Install the following R libraries:  
  
  ```R
  install.packages(c("shiny", "tidyverse", "scales"))
  ```

### Setting Up the App

1. Download the entire repository containing this Shiny app.
2. Place your `.eigenvec`, `.eigenval`, and `popinfo` files in a folder of your choice.
3. Open `PopCanvas-app.R` in RStudio.

## Running the Application

1. Run the Shiny app from RStudio using the following command:
   
   ```R
   shiny::runApp("path/to/app.R")
   ```
2. Upon running the app, the interface will prompt you to select the `.eigenvec` file and the `popinfo.txt` file.
3. Once the files are loaded, the interactive PCA plot will be displayed.

## File Formatting Guidelines

- **Eigenvec File (`.eigenvec`)**:
  
  - Must contain the first two columns as `FID` and `IID`.
  - Additional columns represent the PCA components (PC1, PC2, etc.).

- **Eigenval File (`.eigenval`)**:
  
  - A single-column file with eigenvalues corresponding to each principal component.

- **Popinfo File (`popinfo.txt`)**:
  
  - The first column should match the sample IDs (`IID`) from the `.eigenvec` file.
  - Include relevant metadata such as population, region, or custom labels.
  - Example:
    
    ```
    ID       POP      POPULATION
    HG00119  GBR      Europe
    HG00120  GBR      Europe
    HG00275  CHB      East Asia
    ```

## Application Guide

### UI Controls

- **Title Input**: Set a custom title for the PCA plot.
- **Component Selection**: Choose principal components for X and Y axes.
- **Flip Axes**: Flip the X or Y axis if needed.
- **Plot Type**:
  - **Points**: Display points only.
  - **Text**: Show IDs as text labels.
  - **Labels**: Show population labels.
- **Populations Displayed**: Filter the displayed populations.
- **Population Emphasis**: Highlight a specific population.
- **Group Coloring**: Color the points by a chosen category (e.g., `POPULATION`, `REGION`).
- **Group Centroids**: Enable centroids for each population or group.
- **Legend**: Toggle legend display.

### Interactive Plotting

- **Zoom**: Select an area on the plot using the brush tool and double-click to zoom in.
- **Reset Zoom**: Double-click outside the selected area to reset the view.
- **Brush Info**: View metadata for the brushed points in the `Points Info` panel.

## Saving the Plot

- **Save as PNG**: Download the plot as a PNG file.
- **Save as PDF**: Download the plot as a PDF file.

## Troubleshooting

- Ensure that the `popinfo` file's first column matches the IDs in the `.eigenvec` file.
- Check for missing or incorrectly formatted data in the input files.
- If the app fails to run, make sure all necessary R libraries are installed.
